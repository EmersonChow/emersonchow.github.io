Machine Learning and Data Mining

This is a completed project written between March 2020 and June 2020. In this project, a set of data was given as training data. Working with 2 other students, we created and tested a series of ML models. Written entirely in Python, we tested Neural Networks, KNN classifier, Random Forest, and Gradient Boosting techniques. Additionally, we tried 3 different ensembles of [KNN (weight: 0.8), Random Forest(weight: 0.1), Gradient Boosting(weight: 0.1)], [Random Forest(Weight = 0.5),Gradient Boosting(Weight = 0.4),KNN(Weight = 0.1)], and [Random Forest(Weight = 0.5), Gradient Boosting(Weight = 0.3)/KNN(Weight = 0.1), Neural Networks(Weight = 0.1)]. Through testing, we found that the Random Forest to be the best performing model and the last listed ensemble to be the best with an Area Under Curve - AUC - of 0.79323.
