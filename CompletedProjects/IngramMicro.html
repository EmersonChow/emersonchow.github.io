Ingram Micro: Build an elasticsearch tool for realtime package tracking

This is a completed project that I am worked on with a group of 4 other students. We worked with Ingram Micro to develop a tool using elasticsearch to help them track packages in near real time as they make their way through their warehouses. It ingested data via a series of CSV files to generate an elasticsearch database that customer service representatives can use to find packages and relay information to clients. The application was written in Python using flask. Unfortunately, the code is not publicly available.
